Le SIP

SIP pour Session Initiation Protocol.

  • Protocole standard et ouvert normalisé par l'IETF
  • VoIP mais aussi : IM, Video, Réalité virtuelle, jeux vidéo...
  • En France
    • 2 ISP : Free et SFR fournissent un compte SIP.
    • Il y a aussi des offres via Ippi et Keyyo Mobile.
  • Des providers internationaux.

Exemple classique d'appel (négociation)

Les éléments d'un réseau SIP

  • User agent
  • Registrar (optionnel)
  • Proxy (optionnel)

Exemples de topologies

  • P2P : User agent à user agent
  • Registrar + Proxy ouvert ex : sip2sip.info
  • Gateway vers le réseau PSTN ex : freephonie
  • SIP trunking ex : pbxes.org
  • Avoir son propre server / celui de l'entreprise

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This let's you apply broader style changes, like switching the background.

"blackout"

"soothe"

Additionally custom events can be triggered on a per slide basis by binding to the data-state name.

function linkify( selector ) {
    if( supports3DTransforms ) {
        
        var nodes = document.querySelectorAll( selector );

        for( var i = 0, len = nodes.length; i < len; i++ ) {
            var node = nodes[i];

            if( !node.className || !node.className.match( /roll/g ) ) {
                node.className += ' roll';
                node.innerHTML = '' + node.innerHTML + '';
            }
        };
    }
}
